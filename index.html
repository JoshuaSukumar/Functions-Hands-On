Question 6 :- 

         var x = 21;
         a();
         b();
         function a() {
            x = 20;
            console.log(x);
         };
         function b() {
            x = 40;
            console.log(x);
         };

First output is "20" because the global variable x = 21 is changing inside the a() function, x = 20 becomes a local variable inside the function a() so we declared and initialized the value so it will return the output "20".

Second output is "40" same as the first output we change the variable inside the function b() so "x" became a local variable inside the function and declaration and initialization is also done therefore resulting in "40".

A function can be called before declaration it is called "Function Hoisting" there will be no error

Question 9 :- Guess the Output

         Abc();
         const Abc = function(){
            let value = 20;
            console.log(value);
         }

Uncaught ReferenceError: Cannot access 'Abc' before initialization

Here Abc() is an arrow function, where an arrow function cannot be called before declaration because the arrow function is stored inside the variable. A variable does not return any value before declaration.
